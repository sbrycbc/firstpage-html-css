Display

Mit display: ; können wir festlegen wie sich ein element im zusammenhang mit anderen elementen verhält, der default wert ist display: block; außer text-elementen, die zum beispiel a, b oder em, diese sind im standardfall mit dem wer display: inline; belegt.

display: block;

Mit display: block; sorgen wir dafür, das ein element 100% breit ist, und den ganzen platz der definierten seitenbreite einnimmt, wenn wir also ein weiteres element danach hinzufügen, wird dieses in die nächste zeile rutschen

display: none;

display: none; versteckt ein element komplett, es nimmt dann auch keinen platz mehr auf der html seite ein.

display: inline-block;

display: inline-block; sorgt dafür, das wir block elemente nebeneinander legen können, alle styles wie width und height funktionieren hier immernoch.

Es gibt in css einen bug, der dafür sorgt, das elemente die nebeneinander liegen, bei display: inline-block; ein unsichtbares leerzeichen zwischen ihnen hat, um dieses zu beheben, gibt es ein paar möglichkeiten, die zuverlässigste (standard) lösung, ist, in der zeile wo ein element endet, einen kommentar anzufangen und beim nächsten element das kommentar zu beenden
