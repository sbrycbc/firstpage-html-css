Selektoren

In css nutzen wir selektoren um das element, das wir stylen wollen auszuwählen, ein css kommando ist wie folgt aufgebaut:

  • [element] { = der selektor
  • [eigenschaft]: [wert]; = die option
  • } = das ende des kommandos

Mit komma getrennt, können wir mehrere selektoren auswählen und ihnen die selben kommandos geben.
